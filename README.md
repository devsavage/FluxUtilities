# FluxUtilities
This is an add-on mod for Thermal Expansion. Adding many new blocks/items to Minecraft.  

Requirements
=====
Minecraft Version: 1.7.10  
Forge Version: 10.13.2.1921+  
CoFH Core: 3.0.0RC7+
